# 利率模型

Bend 的利率模型经过校准，以管理流动性风险和优化利用。借款利率来自于利用率 $$U$$。

$$U$$ 是资金池中可用资金的一个指标。利率模型被用来通过用户激励支持流动性来管理流动性风险：

* 当资本可用时：低利率以鼓励贷款。
* 当资本匮乏时：高利率以鼓励偿还贷款和增加存款。
